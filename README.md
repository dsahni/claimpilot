# ClaimPilot

Demo prototype app for an AI-powered auto insurance claims agent.

## Running it locally

```bash
git clone <repo-url>
cd claims_agent
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Log in as **Policyholder** to submit and track claims, or as **Claims Adjuster** to review the AI assessment queue. 

## Notes
Any credentials work — auth is mocked. 

No persistent storage, so all claim data is cleared on refresh.
